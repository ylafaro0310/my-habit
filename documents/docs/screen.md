# 画面仕様

## 画面遷移図
```uml
@startuml
[*] --> 習慣記録画面
習慣記録画面 --> 習慣追加画面
習慣記録画面 --> 習慣編集画面
習慣記録画面 --> 作業記録一覧画面
習慣記録画面 --> 習慣分析画面

習慣追加画面 --> 習慣記録画面: 習慣追加
習慣編集画面 --> 習慣記録画面: 習慣更新 or 削除

作業記録一覧画面 --> 作業記録追加画面
作業記録一覧画面 --> 作業記録編集画面

作業記録追加画面 --> 作業記録一覧画面: 作業記録追加
作業記録編集画面 --> 作業記録一覧画面: 作業記録更新 or 削除


習慣記録画面: do / 習慣記録追加
習慣記録画面: do / 習慣記録削除
習慣記録画面: do / 表示日付選択
習慣記録画面: do / 達成済み表示/非表示切り替え
習慣記録画面: do / 習慣のカテゴリ選択
習慣記録画面: do / 時間帯フィルタ(未実装)


習慣分析画面: do / 習慣記録追加
習慣分析画面: do / 習慣記録削除

習慣追加画面: do / 習慣追加

習慣編集画面: do / 習慣更新
習慣編集画面: do / 習慣削除

作業記録追加画面: do / 作業記録追加

作業記録編集画面: do / 作業記録更新
作業記録編集画面: do / 作業記録削除
@enduml
```
